Important Notes for working with Blender3D

Blender3D has to be started manually inside the container.
To do so, click on the terminal and type one of the following to lines:

$ cd bin/blender
$ ./blender

Convert CAD-Data to URDF

With the Blender3D Plugin Phobos the user is graphically supported to describe the robot, having only CAD-Files, in URDF.
While following these steps, you will get a ROS robot description in an easy and fast way.
