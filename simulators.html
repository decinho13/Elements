Gazebo

Gazebo is a robust physics engine, offers high-quality graphics, and a convenient programmatic and graphical interfaces. It is a well-designed simulator, that makes it possible to rapidly test algorithms, design robots, perform regression testing, and train AI system using realistic scenarios.
(http://gazebosim.org/)


  • Start

Important Notes for working with Gazebo

Gazebo has to be started manually inside the container.
To do so, click either on the terminal or on the Cloud9-Terminal and type one of the following to lines:

gazebo --verbose
rosrun gazebo_ros gazebo

V-REP

V-REP is used for fast algorithm development, factory automation simulations, fast prototyping and verification, robotics related education, remote monitoring, safety double-checking, etc.
Each object/model can be individually controlled via an embedded script, a plugin, a ROS or BlueZero node, a remote API client, or a custom solution. This makes V-REP very versatile and ideal for multi-robot applications.
(http://www.coppeliarobotics.com/)

  • Start

Important Notes for working with V-REP

V-REP has to be started manually inside the container.
To do so, click on the terminal and type in the following to lines:

cd bin/vrep
./vrep
