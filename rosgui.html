ROS GUI-Tools

Use powerful tools to bring your applications to a higher level

Rviz

rviz (ROS visualization) is a 3D visualizer for displaying sensor data and state information from ROS. Using rviz, you can visualize Baxter's current configuration on a virtual model of the robot. You can also display live representations of sensor values coming over ROS Topics including camera data, infrared distance measurements, sonar data, and more. (http://sdk.rethinkrobotics.com/wiki/Rviz)
For more Information have a look on this tutorial webpage.
To use this tool type in following command in Cloud9-Terminal:

$ rviz

Rqt

rqt is a software framework of ROS that implements the various GUI tools in the form of plugins. Over 20 plugins have already been created and more are slated for development.
For more Information have a look on this tutorial webpage.
(wiki.ros.org/rqt)
To use this tool type in following command in Cloud9-Terminal:

$ rqt

MoveIt!

MoveIt is state of the art software for mobile manipulation, incorporating the latest advances in motion planning, manipulation, 3D perception, kinematics, control and navigation. It provides an easy-to-use platform for developing advanced robotics applications, evaluating new robot designs and building integrated robotics products for industrial, commercial, R&D and other domains. (https://moveit.ros.org/)
For more Information have a look on this tutorial webpage.
To use this tool type in following command in Cloud9-Terminal:

$ roslaunch moveit_setup_assistant setup_assistant.launch
